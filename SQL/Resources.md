# 📚 High-Value SQL Resources

Handpicked, authoritative resources to master SQL query writing, performance tuning, relational schema design, and engine internals.

---

## 📖 Official Documentation & Manuals

- **[PostgreSQL Official Documentation](https://www.postgresql.org/docs/)**: The gold standard for open-source database documentation. Excellent details on MVCC, window functions, and indexing mechanics.
- **[MySQL InnoDB Engine Manual](https://dev.mysql.com/doc/refman/8.0/en/innodb-storage-engine.html)**: Essential reading for understanding InnoDB locking, gap locks, undo logs, and transaction isolation.
- **[SQLite Query Planner Guide](https://www.sqlite.org/opdatasets.html)**: Outstanding intuitive explanation of how SQL optimizers use indexes, B-Trees, and loop joins.

---

## 📘 Must-Read Books

- **"Use The Index, Luke!" by Markus Winand**: The single best book on database performance tuning and SQL indexing for developers.
- **"Designing Data-Intensive Applications" (DDIA) by Martin Kleppmann**: Chapters 3 & 7 provide world-class explanations of storage engines (B-Trees vs LSM Trees), transactions, isolation levels, and concurrency anomalies.
- **"SQL for Smarties" by Joe Celko**: Advanced SQL set manipulation, hierarchical data representation, and declarative problem-solving.

---

## 🖥️ Interactive Practice Platforms

- **[LeetCode Database Section](https://leetcode.com/problemset/database/)**: Excellent for practicing medium/hard coding problems (Window Functions, CTEs, Anti-joins).
- **[SQLZoo](https://sqlzoo.net/)**: Beginner to intermediate step-by-step interactive query builder.
- **[PGFiddle / DB Fiddle](https://www.db-fiddle.com/)**: Test queries, execution plans (`EXPLAIN`), and schema builds live in your browser across PostgreSQL, MySQL, and SQLite.
- **[Mode Analytics SQL Tutorial](https://mode.com/sql-tutorial/)**: Real-world data analytics queries and case studies.

---

## 🎥 YouTube Channels & Visual Learning

- **[Hussein Nasser](https://www.youtube.com/@HusseinNasser)**: In-depth architectural discussions on database internals, connection pooling, ACID, B-Trees, indexes, and locking mechanics.
- **[CMU Database Group (Andy Pavlo)](https://www.youtube.com/@CMUDatabaseGroup)**: Carnegie Mellon University's Intro & Advanced Database Systems lectures (15-445/645). Unmatched technical depth into database engine code.

## 📰 Engineering Blogs

- **[Database Internals Substack / Blogs](https://www.databass.dev/)**: Deep technical explorations of query optimizers and storage layout.
- **[AWS Architecture & Database Blog](https://aws.amazon.com/blogs/database/)**: Real-world scaling, Aurora PostgreSQL internals, and multi-region replication architectures.
